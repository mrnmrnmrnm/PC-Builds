<h2>Description</h2>
Today I will explain the PC specifications of my personal computer build. While doing research for my build, I wanted a PC that would be great for gaming, surfing the web, and studying since I do at least one on a daily basis. Since I was on a budget, I was looking for a good price to performance ratio for each component. It is easy to just get the cheapest parts but since I wanted something reliable I made sure to use quality where needed to have a good running system.

<h2>Assembly + Parts Description:</h2>

Before starting make sure to have a clean workspace with ample space to assemble your machine. Idealy you would use an anti-static wrist strap or mat while working to ensure sensitive electronics do not recieve any damage.

<br>

The Parts I used for this build:
- <b>Cooler Master MasterBox NR400 (w/ODD) MicroATX Mid Tower Case</b>

Great budget case that had ample space for motherboard, storage, fans, and GPU.

- <b>Gigabyte B660M DS3H DDR4 Micro ATX LGA1700 Motherboard</b>\

Motherboard that supported my required DDR4 RAM (cheaper option than DDR5), Intel CPU Socket, SATA ports, NVME socket etc. 

- <b>Cooler Master Hyper 212 Black Edition 42 CFM CPU Cooler</b>

Cooler was a great price with great cooling while matching the brand of case and theme of build .

- <b>Intel Core i5-12400F 2.5 GHz 6-Core Processor</b>

CPU was on sale when purchased and will be great for upgrades to the GPU in the future.

- <b>EVGA SC2 HYBRID GAMING GeForce GTX 1080 Ti 11 GB Video Card</b>

GPU was bought used online, it comes from a reputable brand (EVGA). Provides the performance I was looking for and includes built-in liquid cooling instead of air

- <b>TEAMGROUP T-Force Dark Za 32 GB (2 x 16 GB) DDR4-3200 CL16 Memory</b>

Budget pick using a brand with great reviews. Matched the speeds I was looking for at a great price. Also matches theme of build.

- <b>Silicon Power P34A60 512 GB M.2-2280 PCIe 3.0 X4 NVME Solid State Drive</b>

Using as a boot device, NVME speeds would be an upgrade vs the traditional SSD I had before. 

- <b>Western Digital Blue SA510 1 TB 2.5" Solid State Drive</b>

As a drive I already had, this was my primary drive before repurposing it as my addtional drive. It will be used for any files/programs I need saved. 

- <b>Corsair RM650 650 W 80+ Gold Certified Fully Modular ATX Power Supply</b>

I chose this PSU since Corsair is a reputable brand, it fit my budget, and provided more then anough power for my build.

- <b>Apevia C 57.67 CFM 120 mm Fans 5-Pack</b>

These fans were used instead of the fans included with the case. I wanted something to help light up the inside of the PC and provide ambiance for the build.

Assembly and OS:

1. Unpackage Case on your designated workstation and orientate so that the motherboard mounting is facing the ceiling. This helps with setting up the internals.

2. Remove all removable panels on case.

3. If removing or installing addtional fans, install them now into the case making sure to face them the way you want the airflow to go. The fans should come with hardware to install them into the case holes.

4. Unpackage motherboard and place on top of box for assembly.

5. Unpackage CPU and RAM and place them in designated slots on motherboard following the guide included with motherboard, CPU, and RAM.

6. After CPU and RAM are secure, unpackage CPU cooler attaching with included instructions.

7. If you have any NVME drives, this would be a great time to install it on the motherboard following motherboard instructions.

8. Included with the case should be mounting hardware for the motherboard to attach to the case. Make sure it is installed for the motherboard size you are installing.

9. At this point your motherboard should be ready to be installed into the case. First install the I/O shield into the case then slowly lower the motherboard into the case and secure using provided hardware.

10. If adding any addtional SSDs or HDDs that are not NVME, you can mount them into the case following the case instructions. 

11.  Now the GPU can be unboxed and inserted into the PCIe slot. Make sure everything lines up and push it into the slot. the provided hardware will be used to lock in the GPU into the case.

12.  If you are using any other Internal components like a DVD/RW Optical Drives, now would be a great time to install them into the case.

13.  Now the PSU will be inserted into the case and installed following included instructions.

14.  After it is secure, you can now start running power wires from the PSU into the motherboard, GPU, SSDs/HDDs, interal drives, etc. Try to keep everything organized using zip-ties and hiding wires if possible. Make sure to double check each component for any PSU power requirements/inputs.

15.  Now that power is handled, run any wires from the motherboard to fans, storage drives, optical drives, case inputs etc. The motherboard manual is your best friend. Makes sure all connections are secure and in the correct slot. Still try to keep the case organized while plugging things in.

16.  Double check to make sure eveything is plugged in correctly.

17.  Plug in any computer peripherals (mouse, keyboard)  into the motherboard/case and Displays (monitors) into the GPU.

18.  Plug in the power supply into the wall outlet or power outlet strip and make sure the power supply is switch is on.

19.  Try to start your machine!

20.  If everything is working correctly you should see a bios screen come up after a few seconds.

21.  Now to power everything off again and get ready to install an operating system.

22.  If you have an optical disk drive you may be able to use an install disk to install the OS. Just insert the disk into the drive and configre the BIOS to boot from the optical drive. After following the installation wizard, you should be able to setup bios to boot from the drive the OS is now installed on and use the PC!

23. Follow these steps to create a bootable [Windows](https://support.microsoft.com/en-us/windows/create-installation-media-for-windows-99a58364-8c02-206f-aa6f-40c3b507420d) install on a USB drive. If using another OS, there should be similar instructions online on how to create one for that OS.

24. After creating this, Plug in the Drive into a USB port on the PC and set the BIOS to start from the USB drive. After following the installation wizard, you should be able to setup bios to boot from the drive the OS is now installed on and use the PC!
